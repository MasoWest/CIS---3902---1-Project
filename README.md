# CIS---3902---1-Project
Github repo for the CIS 3902 - 1 final dataset project

- The utlized dataset shall be "Phishing Websites" by Mohammad, R. & McCluskey, L. (2012). Phishing Websites [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C51W2X.

- Variables within the notebook consist of traits of phishing websites given numerical indicators (1, -1, etc.)

- Currently, a phishing website is detected if an Is_Phishing_SIte feature detects a 1 in the Results column. 0/-1 Indicates a trait is not present in the website.

- The latest juypter notebook iteration is linked here: https://github.com/MasoWest/CIS---3902---1-Project/blob/main/DataMining_Project_Phishing_Spring_2026_MGW4_3.ipynb

- Basic ai assistance (questioning obscure topics) was utilized.
